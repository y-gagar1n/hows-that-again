# Github Markdown

## Emojis

Галочка - u2714 ✔️
Крест - u2716  ✖️

Чтобы гитхаб превратил символ в картинку, нужно после него писать невидимый сивол uFE0F

На линуксе юникодные символы можно набирить, нажимая Ctrl + Shift + u + <HEX-отображение символа>
